# अंक ज्योतिष भविष्य

एक आधुनिक मराठी अंकशास्त्र वेबसाइट ज्यामध्ये जन्मतारीख आधारित अंक विश्लेषण, जीवनमार्ग, शुभ अंक आणि सविस्तर मार्गदर्शन दिले जाते.

## वैशिष्ट्ये
- जन्मांक, जीवनमार्गांक आणि शुभांक गणना
- सविस्तर अंकशास्त्र माहिती
- डार्क/लाइट थीम
- शोध, कॉपी, शेअर, प्रिंट आणि PDF डाउनलोड
- Express.js सर्व्हर

## चालवण्याची पद्धत
```bash
npm install
npm start
```

आता http://localhost:3000 वर उघडा.
